## Giới thiệu

DS OPEN HUB là kho tài nguyên mở do Đoàn khoa Khoa học Dữ liệu trong Kinh doanh phát triển, nhằm hệ thống hóa tài liệu học tập và tạo không gian chia sẻ tri thức cho sinh viên.

Dự án tập trung vào ba mục tiêu chính:

- Chuẩn hóa nguồn tài liệu học và thực hành theo cấu trúc rõ ràng.
- Tăng khả năng tiếp cận dataset, ebook và project mẫu cho sinh viên.
- Hỗ trợ học tập định hướng ứng dụng qua các case thực tế.

## Truy cập nhanh

- Website: https://hub-dsb-youth.github.io/ds-open-hub/
- Projects: 1. PROJECTS
- Ebooks: 2. EBOOK
- Datasets: 3. DATASET

## Cấu trúc kho dữ liệu

| Thư mục | Mô tả |
|---|---|
| 1. PROJECTS | Dự án thực hành phân tích dữ liệu, Machine Learning, dashboard, Streamlit app |
| 2. EBOOK | Tài liệu tham khảo theo chủ đề: Data Science, AI, Python, SQL, Mathematics, ... |
| 3. DATASET | Bộ dữ liệu phục vụ EDA, modeling, báo cáo và nghiên cứu |

## Cách sử dụng

1. Chọn thư mục theo mục tiêu học tập: project, ebook hoặc dataset.
2. Đọc README tại từng thư mục con để nắm ngữ cảnh và hướng sử dụng.
3. Tải file cần thiết hoặc clone repository để học offline.
4. Truy cập website để điều hướng nhanh bằng giao diện trực quan.

## Đóng góp

Bạn có thể đóng góp theo các hình thức sau:

- Bổ sung dataset có nguồn rõ ràng và mô tả đầy đủ.
- Chia sẻ notebook, report hoặc project có giá trị học tập.
- Cải thiện README, tài liệu hướng dẫn, hoặc sửa lỗi liên kết.

Quy trình gợi ý:

1. Fork repository.
2. Tạo nhánh mới cho thay đổi.
3. Commit với mô tả ngắn gọn, rõ mục tiêu.
4. Mở Pull Request để nhóm quản trị review.
