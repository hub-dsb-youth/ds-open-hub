# DS Open Hub

DS Open Hub là ngân hàng tri thức mở do cộng đồng sinh viên xây dựng, giúp học và thực hành Data Science theo hướng có cấu trúc, dễ truy cập và có thể tái sử dụng ngay.

Trang docs này đóng vai trò website miễn phí cho repository, giúp truy cập nhanh tới:

- Dự án thực hành trong thư mục 1. PROJECTS
- Danh mục dữ liệu học tập trong thư mục 3. DATASET
- Kho ebook tham khảo trong thư mục 2. EBOOK

## Mục tiêu chính

- Chuẩn hóa điểm truy cập tài liệu thay vì duyệt thư mục thủ công trên GitHub
- Giảm thời gian onboarding cho sinh viên mới
- Gom liên kết tới các notebook, báo cáo, dashboard và ứng dụng demo

## Điều hướng nhanh

- [Projects](./projects): tổng hợp các project phân tích và ứng dụng ML
- [Datasets](./datasets): danh mục dataset và gợi ý bài toán
- [Ebooks](./ebooks): nhóm tài liệu theo chủ đề
- [Roadmap](./roadmap): lộ trình học tập đề xuất cho tân sinh viên
- [Blogs](/blog): chia sẻ định hướng ngành và quá trình học

## Giá trị cốt lõi

- Học nhanh: truy cập thẳng vào file notebook, CSV, README mà không cần đào sâu cây thư mục.
- Học sâu: có lộ trình tài liệu nền tảng đến nâng cao qua tab Ebooks.
- Học thực tế: mỗi project gắn với bài toán thực tế và hướng ứng dụng.

## Lưu ý sử dụng

- Website này ưu tiên Link, don't embed: liên kết về nguồn gốc trong repository thay vì sao chép toàn bộ nội dung.
- Các app chạy bằng Streamlit không host trực tiếp trên GitHub Pages; nên triển khai riêng và gắn link vào mục Projects.
