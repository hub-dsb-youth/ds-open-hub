# 1.4 Bank Churn Analysis

## Tiếng Việt

### Mục tiêu

- Phân tích churn khách hàng ngân hàng theo nhân khẩu học và hành vi.
- Tạo dashboard trực quan phục vụ quyết định giữ chân khách hàng.

### Thành phần chính

- Dữ liệu raw và processed
- Notebook tiền xử lý dữ liệu
- Dashboard Power BI
- Report tổng hợp insight

### Công nghệ

- Python (pandas, numpy)
- Power BI
- DAX

### Giá trị học tập

- Làm sạch dữ liệu và tạo feature theo bối cảnh ngân hàng.
- Trực quan hóa churn theo geography, age, balance, credit score.

## English

### Objective

- Analyze bank customer churn by demographic and behavioral factors.
- Build interactive dashboards for customer retention decisions.

### Main assets

- Raw and processed datasets
- Data preparation notebook
- Power BI dashboard
- Insight report

### Stack

- Python (pandas, numpy)
- Power BI
- DAX

### Learning value

- Apply feature engineering in a banking context.
- Visualize churn patterns across geography, age, balance, and credit score.
