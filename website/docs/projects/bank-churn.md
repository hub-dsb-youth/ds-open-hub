## Tiếng Việt

### Mục tiêu

- Phân tích churn khách hàng ngân hàng theo nhân khẩu học và hành vi.
- Tạo dashboard trực quan phục vụ quyết định giữ chân khách hàng.

### Thành phần chính

- Dữ liệu raw và processed
- Notebook tiền xử lý dữ liệu
- Dashboard Power BI
- Report tổng hợp insight

### Công nghệ

- Python (pandas, numpy)
- Power BI
- DAX

### Giá trị học tập

- Làm sạch dữ liệu và tạo feature theo bối cảnh ngân hàng.
- Trực quan hóa churn theo geography, age, balance, credit score.
