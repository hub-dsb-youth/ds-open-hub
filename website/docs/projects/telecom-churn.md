## Tiếng Việt

### Mục tiêu

- Phân tích nguyên nhân churn trong ngành viễn thông.
- Kết hợp xử lý dữ liệu bằng Python, dashboard Power BI và mô hình dự đoán.

### Thành phần chính

- Dữ liệu gốc và dữ liệu đã làm sạch
- Notebook tiền xử lý và notebook dự đoán
- Model logistic regression
- Ứng dụng Streamlit cho dự đoán theo input người dùng

### Công nghệ

- Python, pandas, numpy, scikit-learn
- Power BI (DAX + dashboard)
- Streamlit

### Giá trị học tập

- Quy trình end-to-end từ data prep đến deployment demo.
- Học cách diễn giải churn driver theo góc nhìn kinh doanh.
